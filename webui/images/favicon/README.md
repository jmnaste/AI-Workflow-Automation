# WebUI Favicon Assets

Place generated favicon and PWA assets here (see branding.md for generator suggestions).

Minimum recommended files:
- favicon.ico
- favicon-16x16.png
- favicon-32x32.png
- apple-touch-icon.png
- site.webmanifest

Update `index.html` link tags if paths differ. For now, `index.html` references `/images/favicon/favicon-32x32.png`.
